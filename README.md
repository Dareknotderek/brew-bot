# brew-bot

Bot that spits out brew recipes for n-gallon batches. Made originally for [wallbrew.com](https://wallbrew.com/)

## Installation

Production build:
```
lein prod-build
```

Dev/Figwheel build:
```
lein dev-build
lein figwheel
```

Then open `index.html` in the browser of your choice.
